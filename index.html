Capabilities

What PHAROS Does

A complete adaptive optics processing chain with dual classical and ML paths.

⚛

Classical Pipeline

Full SVD-based wavefront reconstruction via Zernike modal decomposition. Dark subtraction, flat-field correction, threshold centroiding, and Noll-based turbulence estimation out of the box.

⚡

ML-Accelerated Path

Three trained neural networks as drop-in replacements: CentroidCNN for sub-pixel spot localisation, WavefrontMLP for slope-to-Zernike reconstruction, and TurbulenceANN for r₀ and τ₀ estimation.

⏰

Real-Time Control Loop

Producer-consumer architecture with separate capture and processing threads. Queue-based frame management with latency monitoring and graceful frame-drop handling.

📊

Benchmarking Suite

End-to-end latency profiling, RMS wavefront error, Strehl ratio computation, and phase structure function validation. Classical vs ML comparison tables.

📦

Model Export

Export trained models to TorchScript and ONNX for deployment. GPU-free CPU inference path for embedded and edge scenarios.

Deep Dive

How It Works

Each pipeline stage maps directly to the physics of atmospheric wavefront sensing.

1

Frame Preprocessing

Raw SH-WFS frames are loaded and normalised to float32. A dark frame is subtracted to remove sensor bias, then a flat-field correction accounts for pixel-to-pixel sensitivity variation. An optimised preprocess_fast path uses precomputed correctors for real-time operation.

2

Centroid Extraction

Each microlens in the 10×10 array focuses its portion of the wavefront onto the detector. The spot displacement within each sub-aperture encodes the local wavefront slope. Three methods available: Centre-of-Mass, Threshold CoM (sigma-clipped), and 2D Gaussian fitting. A vectorised batch path processes all sub-apertures simultaneously.

3

Slope Assembly

Centroid deviations from reference positions are converted to wavefront slopes in radians by dividing by the lenslet focal length. Slopes are assembled into a single vector for reconstruction.

sx = Δx / f    [radians]
4

Wavefront Reconstruction

The slope vector is multiplied by a pseudo-inverse reconstructor matrix R = D⁺ (computed via truncated SVD of the Zernike interaction matrix) to recover the Zernike modal coefficients. The phase map is then synthesised as a linear combination of Zernike polynomials.

c = R · s     W(x,y) = Σ cj · Zj(x,y)
5

Turbulence Characterisation

From the time-series of Zernike coefficients, the Fried parameter r₀ is estimated using the Noll (1976) variance formula. Coherence time τ₀ is extracted by fitting the temporal power spectral density of the slope measurements to the expected f-8/3 Kolmogorov spectrum.

Var(Zj) = 0.2944 · Δj · (D/r₀)5/3 · λ²
6

DM Actuator Commands

The reconstructed phase map is conjugated and the coupling matrix equation I · a = −W is solved via least-squares to compute the 97-actuator stroke vector. Commands are clipped to the physical stroke range (±5 μm) and dispatched to the deformable mirror.

a = I⁺ · (−Wflat)

Quick Start

Getting Started

Six steps from clone to closed-loop simulation.

1 Install Dependencies ▼
$ pip install -r requirements.txt $ pip install -e .
2 Generate Synthetic Data ▼
$ python data/sim/generate_sim_data.py $ python data/sim/generate_kolmogorov_data.py
3 Run the Classical Pipeline ▼
$ python pharos/pipeline_classical.py
4 Train ML Models ▼
$ python ml/reconstructor/dataset.py --generate --n-samples 10000 $ python ml/turbulence_ann/dataset.py --generate --n-samples 50000 $ python ml/train_all.py
5 Closed-Loop Simulation ▼
$ python simulate_loop.py --n-frames 1000 --mode classical $ python simulate_loop.py --n-frames 1000 --mode ml
6 Run Benchmarks ▼
$ python benchmarks/benchmark.py --n-frames 200
